# 安全策略

## 报告安全漏洞

如果你发现了安全漏洞，**请不要公开提交 Issue**。请通过以下方式私下报告：

- **邮箱**: markcell@outlook.com
- **GitHub**: 在仓库中创建一个安全通告

## 响应时间

我们会在 48 小时内确认收到你的报告，并在 1 周内给出初步评估。

## 安全承诺

- 有后台进程（MCP 守护进程 daemon/mcp-daemon.js，仅启动时不安装 cron/launchd/systemd）、无自升级、无 HTTP 服务（使用 Unix Socket）、无凭据存储、无外部通信（仅在用户明确发起请求时使用）、无埋点/遥测
- 所有依赖最小化（仅 @xenova/transformers）
- 本地存储优先，所有用户数据归属用户

## 受支持的版本

| 版本 | 支持状态 |
|------|---------|
| 1.x | ✅ 活跃支持 |
| < 1.0 | ❌ 不再支持 |
