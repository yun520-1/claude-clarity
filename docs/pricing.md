## 常见问题

### 一般问题

**Q: 免费方案可以商用吗？**  
A: 不可以。免费方案仅限个人学习和实验使用。商业用途请选择 Pro 或 Enterprise 方案。

**Q: 可以升级或降级吗？**  
A: 可以。你可以随时在 Dashboard 中更改方案。降级将在下个计费周期生效。

**Q: 支持哪些支付方式？**  
A: 我们支持信用卡 (Visa, MasterCard, Amex)、PayPal、银行转账 (Enterprise)。

**Q: 可以退款吗？**  
A: Pro 方案支持 14 天无理由退款。Enterprise 方案按合同约定。

### 技术问题

**Q: API 速率限制是多少？**  
A: Free: 10 次/分钟, Pro: 100 次/分钟, Enterprise: 可定制。

**Q: 支持哪些编程语言？**  
A: 我们提供 TypeScript/JavaScript SDK。其他语言可通过 REST API 调用。

**Q: 数据如何存储？**  
A: 默认使用我们的云服务。Enterprise 客户可选择私有化部署。

**Q: 支持哪些语言的情绪识别？**  
A: 当前支持中文和英文。其他语言正在开发中。

### 临床验证

**Q: Clarity 有临床验证吗？**  
A: **不，没有。** 本文档是早期设计概念草案，临床验证相关内容为假设性构想。Clarity 是开源认知引擎，不提供任何临床诊断或治疗功能。

**Q: 可以用于医疗诊断吗？**  
A: 不可以。Clarity 不是医疗设备，不能用于诊断或治疗。仅供研究和辅助用途。
